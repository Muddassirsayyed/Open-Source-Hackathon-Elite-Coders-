### Features Showcased in the Video:
1. **Interactive Landing Page**: Sleek design with responsive translation toggle (English / Hindi / Marathi).
2. **Nearby Professional Live Map**: Real-time Geolocation tracking and distance scoring using the Haversine formula.
3. **AI Chatbot**: Intelligent NLP query matching for discovering and booking professionals through natural chat.
4. **Emergency Service Dispatch**: Pulse button that instantly matches the user with the closest online professional.
5. **Real-time Status Sync**: Socket.io notifications when bookings update from Pending -> Accepted -> Completed.
6. **Unified Portal**: Seamless dashboards for users and admin management.
